# 🏦 Bank Management System

A Bank Management System developed using Java and the Spring Framework to handle basic banking operations. This project demonstrates core backend development concepts such as dependency injection, layered architecture, and database integration.

---

## 🚀 Features

- Create and manage bank accounts  
- Deposit and withdraw money  
- View account details  
- Basic transaction handling  
- Structured backend using Spring Framework  

---

## 🛠️ Tech Stack

- Java  
- Spring Framework (Core / MVC)  
- MySQL  

---

## 🏗️ Project Structure

- Controller Layer – Handles user requests  
- Service Layer – Business logic implementation  
- Repository Layer – Database interaction  
- Model Layer – Entity classes  

---

## 🎯 Purpose

The goal of this project is to understand and implement core concepts of the Spring Framework, including dependency injection and MVC architecture, while building a real-world banking application.
